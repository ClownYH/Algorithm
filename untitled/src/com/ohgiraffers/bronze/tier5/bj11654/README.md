# 아스키 코드

### 문제

---
알파벳 소문자, 대문자, 숫자 0-9중 하나가 주어졌을 때, 주어진 글자의 아스키 코드값을 출력하는 프로그램을 작성하시오.

### 입력

---
알파벳 소문자, 대문자, 숫자 0-9 중 하나가 첫째 줄에 주어진다.

### 출력

---
입력으로 주어진 글자의 아스키 코드 값을 출력한다.

예제 입력 1<br>
A<br>
예제 출력 1<br>
65<br>
예제 입력 2<br>
C<br>
예제 출력 2<br>
67<br>
예제 입력 3<br>
0<br>
예제 출력 3<br>
48<br>
예제 입력 4<br>
9<br>
예제 출력 4<br>
57<br>
예제 입력 5<br>
a<br>
예제 출력 5<br>
97<br>
예제 입력 6<br>
z<br>
예제 출력 6<br>
122<br>
