# 숫자의 합

### 문제

---
N개의 숫자가 공백 없이 쓰여있다. 이 숫자를 모두 합해서 출력하는 프로그램을 작성하시오.

### 입력

---
첫째 줄에 숫자의 개수 N (1 ≤ N ≤ 100)이 주어진다. 둘째 줄에 숫자 N개가 공백없이 주어진다.

### 출력

---
입력으로 주어진 숫자 N개의 합을 출력한다.

예제 입력 1<br>
1<br>
1<br>
예제 출력 1<br>
1<br>
예제 입력 2<br>
5<br>
54321<br>
예제 출력 2<br>
15<br>
예제 입력 3<br>
25<br>
7000000000000000000000000<br>
예제 출력 3<br>
7<br>
예제 입력 4<br>
11<br>
10987654321<br>
예제 출력 4<br>
46<br>
